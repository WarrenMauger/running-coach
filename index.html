📋 One-time Setup Checklist

Complete these steps once and syncing will be fully automatic from then on.

  • Make the Google Sheet publicly readable. Open your Warren Training Log, click Share → Anyone with the link → Viewer. This lets the dashboard read it without needing login.
  • Set up IFTTT (if not done yet). Go to ifttt.com → Create → If This: Strava "New activity" → Then That: Google Sheets "Add row to spreadsheet" → Spreadsheet name: Warren Training Log. In the Formatted Row field use: {{OccurredAt}} ||| {{ActivityName}} ||| {{ActivityType}} ||| {{DistanceInKilometers}} ||| {{DurationInMinutes}} ||| {{AveragePaceInMinutesPerKilometer}} ||| {{AverageHeartRate}} ||| {{ElevationGainedInMeters}} ||| {{CaloriesBurned}} ||| Strava
  • Confirm Garmin → Strava sync is active. In Garmin Connect app: More → Settings → Connected Apps → Strava. If it's there and enabled, every Garmin run auto-appears on Strava, which triggers IFTTT.
  • Test it. Log a short run on Garmin. Within a few minutes it should appear on Strava, then in your Google Sheet, then click Sync Now above and it'll land in this dashboard automatically.
